# ホームページ再構築デザイン案（2026-07）

www.juchou-group.com のトップページを再設計した4つのモックアップ。**既存サイトのコードには一切手を入れていない**（docs配下の提案資料のみ）。オーナーが1案を選定したら、その案を `client/src/pages/Home.tsx` へ移植する。

## 見方

各HTMLは自己完結（CSS/JSインライン）。`img/` の写真を相対参照しているので、このフォルダごとブラウザで開けばそのまま動く。

```bash
# ローカルプレビュー例
cd docs/homepage-redesign && npx http-server -o
```

### 案E・Fについて（第2世代 = 「新しい顔」シリーズ）

写真素材を一切使わず、ビジュアルをすべてCanvas 2D/SVGで**プログラム生成**した案。素材の権利・CDN失効・撮影コストから完全に自由で、画面サイズに常にフィットする。QRコード2枚のみ`img/`を参照。

## 全案共通の改良点（現行サイト比）

- **SEO強化**: JSON-LD（`Electrician` LocalBusiness + `FAQPage`）、FAQセクション新設（対応エリア/依頼可能な工事/未経験応募/協力会社取引の4問）、h1〜h3の正しい見出し階層、対応4都県名の明記
- **画像のローカル化**: 現行サイトは有効期限付きの Manus CDN 署名URL（2027年頃失効）に依存。本フォルダの `img/` に退避済みなので、移植時はこれを使うこと
- **パララックス実装**: rAF + translate3d のみ（iOSで壊れる `background-attachment: fixed` 不使用）、`prefers-reduced-motion` 対応、375px〜1920pxレスポンシブ
- 縦書き表現は writing-mode ではなく1文字ずつの縦積み実装（Linux/Android系フォントの縦組みメトリクス欠如による文字重なりを回避）

## 注意

- 採用/問い合わせのCTAは本番の `/recruit` `/contact` への絶対URLリンク。移植時は wouter の `Link` に置換する
- フォントはシステムフォントスタックのみ（Google Fonts不使用）。移植時に現行の Zen Kaku Gothic New / Space Grotesk を使うかは選定後に判断
