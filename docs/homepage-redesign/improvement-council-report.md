# 充寵グループ新HP「整線」改善提案書
## エージェント審議会（分析4名・発案3名・審査3名）統合レポート — 2026-07-10

## 1. 現状診断の要点

10名のエージェントが独立に分析した結果、**弱点は3箇所に完全に合意**しました。

1. **最初の3秒が静止画** — 開いた瞬間に動くものがなく、「3D・PARALLAX」の武器がスクロールするまで一切見えない
2. **中盤に暗黒のデッドゾーン** — 最大の造形的見せ場である「同心円90°ベンド」が何の演出もなく素通りされ、真っ黒な余白が離脱を誘う
3. **結末が軽い** — 1ページかけて幹線を追わせた対価が、左隅の小さな端子台のランプ点灯だけ

さらに強み側の合意: 「6色ケーブル×整線」は既に**所有可能なブランドモチーフ**として機能しており、壊さず深めるべき資産である。

## 2. 【最優先提案】合意パッケージ「一台のカメラと一本の電流」

3名の審査員全員のTop3に入った案を組み合わせた、1本の物語としての改善パッケージです。

### 起 — 「結束の一拍」ロード演出（審査員3名全員が高評価・技術リスク最小）
ページを開いた瞬間、6本のケーブルが左のベンドからラック上を走り、結束バンドが順に「パチン」と締まって静止 → 「充寵グループ」がマスクリビール（1.5秒以内・スキップ可）。
同時に**「走らせて、最後にカチッと締める」動きを全UIの共通言語**（統一イージング `--ease-tie`）として規定し、見出しの出現からボタンのホバーまで、サイト中の動きが同じ「職人の手癖」で終わるようにする。
→ *「最初の3秒が静止画」問題を最小コストで解消し、以降の全演出の品格を規定する土台。*

### 承 — スクロール＝通電（費用対効果最良）
幹線の描画先端に**発光ヘッド（通電の先端）**を常駐させ、スクロール速度に応じて輝度と尾が変化。速く回せば光が強く走り、止まれば静かに呼吸する。通過中のセクション見出しがふわっと照らされる。
→ *ユーザーのスクロール操作そのものが「電気を流す行為」になり、業種と操作が結合する。中盤の黒地が「配線中」の意味を持つ。*

### 転 — 同心円90°ベンドのピン留めシーン（本提案の署名ショット）
ベンド部を約150vhピン留めし、スクロール進捗に応じて6本が外周から順にRを描いて曲がり、**視点（カメラ）もケーブルと一緒に曲がって落ちていく**。「BEND R — CONCENTRIC 90°」の図面注記が浮かぶ。
→ *「曲げ」は職人の腕が最も出る工程。同心円のRが揃った絵は雑な会社には作れない——規律をカメラワークで体感させる、キー用語「3D・PARALLAX」への主回答。*
※審査で意見が割れた点: カメラ回転を8度に抑える安全設計（オーナー代弁者・実装リアリスト支持）か、90度回り切る（awwwards審査員「8度で妥協するならやらない方がいい」）か。**実装時にA/B両方を作って実機で選ぶことを推奨。**

### 結 — 端子台マクロの4拍フィナーレ（全審査員Top3）
最終セクションをピン留め（120vh・スキップリンク常設）し、端子台を画面中央に大きく据えて、スクロールで4拍を刻む:
**①フェルール（棒端子）が挿入され → ②ネジ頭が回転して締まり → ③ランプが6色順に点灯 → ④「結線、完了。」が立ち上がり、CTAに通電パルス。**
→ *「ネジを自分のスクロールで締める」疑似体験は、元請けには本物感、求職者には「この仕事をする自分」として刺さる。締結＝契約のメタファーがCTA直前に来る、演出とコンバージョンの一致。*

**実装規模: 正味3〜5日相当。全てvanilla JS（sticky＋進捗scrub＋SVG transform）で、モバイル60fps・reduced-motion対応を維持。**

## 3. 【次点提案】

### A. 三言語切替「三芯切替」（審査員全員が価値を認めつつ「別プロジェクト」と判定）
ヘッダーに日本語・Português・English のトグルを3本の芯線ラベル風UIで常設。切り替えると選択言語の色のパルスが幹線を流れ、通過した順にテキストが差し替わる——「言語を選ぶ＝その色の電気を通す」儀式。ブラジル人求職者の直帰を止める採用の実利と、世界に前例のない演出が重なる強い案。**ただし翻訳監修が済むまで公開しない・SEO効果は限定的（本命は将来の3ページ静的化）という条件付き。**

### B. WebGL真3Dヒーロー「通電ドリーバック」（審査が最も割れた案）
ケーブル被膜の超マクロ接写から全景へ引くカメラ＋カーソルが実光源になる生シェーダー3D。awwwards審査員は「世界基準で目に留まる唯一の案・SOTD級」と最高評価、実装リアリストは「低速Android破綻と2レンダラ保守で却下」、オーナー代弁者は「二段構え（Canvas版を先に出荷し、余力ある端末のみWebGL昇格）なら保留」。**→ 合意パッケージの成功後に挑戦する第2フェーズ候補。**

## 4. 見送り判断

- **拍節グリッド TIE INTERVAL 192**（スクロールが刻む無音のリズム）— 思想は美しいが「誰も気づかない改善」の公算大。速度比の整数比化など不可視の原理としてのみ採用
- **検電カーソル** — カスタムカーソルは主戦場のスマホに存在せず2019年的。「6色⇔工事種別ラベル＋タップでセクションジャンプ」の部分だけ通電演出に併合
- **200vh版フィナーレ** — CTA直前に2画面分の儀式は商売の導線として本末転倒。120vh版に統一
