# 参考資料メモ

## 1. 作業員名簿PDF（OkiSanaeMeibo.pdf / お(1).pdf）
- ヘッダー: 充寵グループ ロゴ + 「作業員名簿」タイトル
- フィールド:
  - ナマエ（カナ）、名前（漢字）、経験年数
  - 生年月日、血液型、性別
  - 本籍地
  - 在留資格、在留カード期限、在留カード番号
  - 郵便番号、住所
  - 電話番号
  - 健康診断日、血圧
  - 国民健康保険（記号・番号）
  - 国民年金
  - 労災保険番号
  - マイナンバー
  - キャリアアップ番号
  - 緊急連絡先（ナマエ、名前、続柄、住所、電話番号）
  - 保有資格及び免許テーブル（取得済み資格名、取得年月日、資格取得番号）最大12行

## 2. 出面表PDF（エス・ウェイホールディングス出面.pdf）
- ヘッダー: 充寵グループ ロゴ + ウォーターマーク
- 現場名: 品川4街区 マリオットホテル
- 年月: 2024年10月
- テーブル構造:
  - 行: 最大10名の作業員（番号、ナマエ（カナ）、氏名（漢字））
  - 各作業員に「出勤」行と「残業」行
  - 列: 日付1〜31 + 曜日 + 合計
  - 出勤マーク: ○（出勤）、✕（休み）、△（半日）
  - 最下行: 合計（日ごとの出勤人数合計、総合計）

## 3. 請求書（IMG_8095〜IMG_8100 スクリーンショット）
- 請求書作成画面:
  - 取引先（必須）、敬称
  - テンプレート: 自社情報/振込先、社印・ロゴ
  - 件名、請求書番号（必須）、枝番
  - 請求日（必須）、入金方法、入金期日
  - 明細: 摘要、数量、単位（個/式）、単価、消費税（10%/0%）
  - 源泉徴収トグル
  - 小計（税別）、消費税、合計
  - 備考
- 請求書プレビュー:
  - 取引先情報（社名、郵便番号、住所、担当者）
  - 自社情報（充寵グループ、ロゴ、社印、住所）
  - 請求日、請求書番号、登録番号（T6810341010660）
  - 明細テーブル（摘要、数量、単位、単価、明細金額）
  - 内訳（10%対象、0%対象）
  - 振込先（三井住友銀行 厚木支店、普通口座 9053636、口座名義 ジュウチョウグループ オオキ ミツル）

## 4. 単価管理の要件
- 管理者が従業員ごとに設定:
  - 先方単価（受注）: 先方が充寵グループに支払う単価
  - 支払単価（従業員へ）: 充寵グループが従業員に支払う単価
- 現場ごと or 従業員ごとの単価設定

## 5. 印鑑アップロード
- 請求書用の印鑑（従業員個人の印鑑）
- マイプロフィールに印鑑アップロード欄を追加
